Sell Directly

Description

-You can sell directly through purchase module.
First: entre product in inventory module with cost and sales price.

-Second: open requests for quotation from purchase module and select product.
Unit price in purchase means cost per unit which has been registered for the product before.
If you want to put custom serial number write it through document number.

-Then press on sell directly if you want to sell directly through purchase module.

-When you press on sell directly, sales order will appear.
Unit price in sales order means sales price which has been registered for the product before.

-Confirm it ,then order appear automatically in purchase order and sales order.

-Purchase order

-Sales order

-If you enter the same serial through document number Odoo refuse to enter it.
